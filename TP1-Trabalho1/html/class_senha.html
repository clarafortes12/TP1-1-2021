Detailed Description

Padrão para representação da Senha.

Author
Leandro Bottecchia - 180145169

Regras de formato:

  • Senha é válido caso tenha formato XXXXXXXX.
  • Senha é composta por 8 caracteres.
  • Cada caractere "X" é letra (A-Z ou a-z), dígito (0-9) ou caractere especial.
  • Caractere especiais são: "!", "@", "#", "$", "%", "&" ou "?".
  • Não existe caracter repetido.
  • Existe pelo menos uma letra (maiúscula ou minúscula), um dígito e um caractere especial

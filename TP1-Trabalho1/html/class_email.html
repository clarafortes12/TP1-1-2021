Detailed Description

Padrão para representação do email.

Author
Clara Fortes - 190017503

Regras de formato:

  • email é válido caso tenha formato parte-local@domínio.
  • parte-local é composta por até 64 caracteres.
  • domínio é composto por até 255 caracteres.
  • não tenha ponto nem no ínicio e nem no fim
  • não tenha pontos juntos
  • não tenha mais de um @
